# React Banka Uygulaması

Bu proje, React ve `useReducer` kullanarak geliştirilmiş temel bir bankacılık arayüzüdür. 

# Özellikler:
Bakiye Yönetimi: Para yatırma, çekme ve sıfırlama işlemleri.
Akıllı Uyarılar: Bakiye eksiye düştüğünde renk kırmızıya döner ve uyarı mesajı çıkar.


# Kullanılan Teknolojiler:
React (JSX)
CSS 
JavaScript
